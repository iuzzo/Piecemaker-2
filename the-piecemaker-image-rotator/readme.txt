== Description ==

The Piecemaker 2 WordPress Plugin is the perfect way to dynamically display your images, videos and .swf files on any post or page.

The Piecemaker is a Flash-based 3D Image Rotator. Use the intuitive plugin settings panels to manage multiple Piecemaker galleries (dimensions, images, transitions, links, effects) and display them within blog posts and pages.

== Installation ==

1. Upload the `piecemaker` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload your assets (images, video, flash files) via the menu Piecemaker > Assets 
4. Once your assets are uploaded it is time to create a Piecemaker rotator. Go to Piecemaker > Piecemakers and click 'Add New Piecemaker'. Check all the options and click the 'Add Piecemaker' button. 
5. Having created a new Piecemaker you you now have to add Slides and Transitions. Go to Piecemaker > Piecemakers and navigate to the relevant menu using the icons next to your slider. 
6. To add your Piecemaker into a post or page simply type `[Piecemaker id='your_id'/]` where 'your_id' is the id of the Piecemaker (displayed in the Piecemakers section).
7. That's it!
